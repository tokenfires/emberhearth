# AgentArk: Distilling Multi-Agent Intelligence into a Single LLM Agent

- **Paper ID:** 2602.03955
- **Authors:** Yinyi Luo, Yiqiao Jin, Weichen Yu, Mengqi Zhang, Srijan Kumar, Xiaoxiao Li, Weijie Xu, Xin Chen, Jindong Wang
- **Published:** February 3, 2026
- **Source:** arXiv
- **URL:** https://arxiv.org/abs/2602.03955
- **Relevance:** Multi-agent architectures
- **Referenced in:** [Distill 5 AI Agents Into One](../youtube/discoverai/2026-02-07-distill-5-ai-agents-into-one.md)

## Why This Matters for EmberHearth

Directly applicable to EmberHearth's deployment constraints. If multi-agent debate produces better reasoning, this framework shows how to distill that intelligence into a single model — reducing latency and cost for the always-on assistant use case.

## Key Findings

- Proposes three hierarchical distillation strategies: reasoning-enhanced fine-tuning, trajectory-based augmentation, and process-aware distillation
- Shifts computational burden from inference to training, preserving single-agent efficiency
- Distilled models exhibit strong reasoning and self-correction comparable to multi-agent systems
- Enhanced robustness and generalization across diverse reasoning tasks
- Uses process reward models (PRM) and GRPO optimization for the distillation process
