# Know Thy Enemy: Securing LLMs Against Prompt Injection via Diverse Data Synthesis and Instruction-Level Chain-of-Thought Learning

- **Paper ID:** 2601.04666
- **Authors:** Zhiyuan Chang, Mingyang Li, Yuekai Huang, Ziyou Jiang, Xiaojun Jia, Qian Xiong, Junjie Wang, Zhaoyang Li, Qing Wang
- **Published:** January 8, 2026
- **Source:** arXiv
- **URL:** https://arxiv.org/abs/2601.04666
- **Relevance:** Prompt injection defense

## Why This Matters for EmberHearth

Relevant to EmberHearth's Tron security layer. InstruCoT synthesizes diverse training data with instruction-level chain-of-thought fine-tuning, enabling LLMs to identify and reject malicious instructions regardless of source or context position.

## Key Findings

- Proposes InstruCoT: diverse data synthesis + instruction-level chain-of-thought fine-tuning
- Enables LLMs to identify and reject malicious instructions regardless of source or position
- Evaluated across three dimensions: Behavior Deviation, Privacy Leakage, Harmful Output
- Significantly outperforms baselines across four LLM models
- Maintains utility performance while improving security
