# DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching

- **Paper ID:** 2602.06039
- **Authors:** Yuxing Lu, Yucheng Hu, Xukai Zhao, Jiuxin Cao
- **Published:** February 5, 2026
- **Source:** arXiv
- **URL:** https://arxiv.org/abs/2602.06039
- **Relevance:** Multi-agent architectures
- **Referenced in:** [8B Outperforms GPT 120B on Multi-Agents](../youtube/discoverai/2026-02-10-8b-outperforms-gpt-120b-on-multi-agents.md)

## Why This Matters for EmberHearth

Relevant to EmberHearth's agent coordination architecture. Shows that dynamic, semantically-matched communication graphs outperform fixed topologies by +6.2% — smaller models with smart routing can beat larger models with static patterns.

## Key Findings

- Manager-guided framework reconstructs sparse directed communication graphs each round
- Agents output lightweight query (need) and key (offer) descriptors for semantic matching
- Consistently outperforms strongest baseline by avg. +6.2% across code and math benchmarks
- Works across four different LLM backbones
- Produces interpretable coordination traces via evolving communication graphs
